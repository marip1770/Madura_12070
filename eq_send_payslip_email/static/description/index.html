Send Payslip by Email

Key Features:

  • Allow you to send email of Payslip to employee.
  • Allow you to send multiple employee payslip at same time.
  • You can send payslip email from Payslip Batches.
  • User can change the content of payslip email.

Overview

  • From Payslip Form view, There is button for send payslip by email. When you click on Send by Email button one pop-up will open.
  • In opened pop-up, there is details like employee work email as recipients, subject, email content, pdf of payslip as attachment.
  • PDF report format of attached payslip.
  • For mass mail of payslip, Go to Payslip list view, select the records, select option Send by Email from top Action menu. When user click on Send by Email option, one pop-up window will open.
  • In opened pop-up window, there is button for send payslip.
  • From Payslip Batches form view there is button for sending payslip email.
